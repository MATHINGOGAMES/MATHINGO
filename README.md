# 🐊 Mathingo - منصة ماتينجو التعليمية

تطبيق تعليمي تفاعلي للأطفال يهدف إلى تنمية المهارات الذهنية والحسابية من خلال اللعب.

## 🚀 المميزات (Features)

- **15 لعبة متنوعة**: تشمل الحساب، المنطق، الألوان، الأشكال، والذاكرة.
- **نظام نقاط**: يحفز الطفل على الاستمرار وتحقيق إنجازات جديدة.
- **تصميم تفاعلي**: واجهات ملونة وجذابة مع تمساح "ماتينجو" الودود.
- **تنمية مستدامة**: هيكلية برمجية تسمح بإضافة المزيد من الألعاب بسهولة.

## 🛠️ التقنيات المستخدمة (Tech Stack)

- **React.js** (Frontend Library)
- **Vite** (Build Tool)
- **CSS3** (Animations & Styling)
- **JavaScript ES6+**
